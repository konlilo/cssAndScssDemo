將網格劃分爲區域模板
        你可以將網格中的一些單元格組合成一個區域（area），併爲該區域指定一個自定義名稱。 
        可以通過給容器加上 grid-template-areas 來實現：
        
        grid-template-areas:
          "header header header"
          "advert content content"
          "advert footer footer";
        上面的代碼將網格單元格分成四個區域：header、advert、content 和 footer。 
        每個單詞代表一個單元格，每對引號代表一行。
        

        更改模板，使 footer 區域跨越整個底部行。 定義區域現在不會有任何可視效果。 
        稍後，你將使用一個區域來查看它的工作方式。
