使用 CSS 網格單位來更改列和行的大小
        在 CSS 網格中，可以使用絕對單位（如 px）或相對單位（如 em）來定義行或列的大小。
        
        下面的單位也可以使用：
        
        fr：設置列或行佔剩餘空間的比例，
        
        auto：設置列寬或行高自動等於它的內容的寬度或高度，
        
        %：將列或行調整爲它的容器寬度或高度的百分比，
        
        以下代碼爲右側預覽區中的效果：
        
            grid-template-columns: auto 50px 10% 2fr 1fr;
        
        這段代碼添加了五個列。 
        第一列的寬與它的內容寬度相等；
        第二列寬 50px；第三列寬是它容器的 10%；
        最後兩列，將剩餘的寬度平均分成三份，第四列佔兩份，第五列佔一份。
生成一個包含三列的網格，每列寬度分別爲：1fr、100px、2fr。
