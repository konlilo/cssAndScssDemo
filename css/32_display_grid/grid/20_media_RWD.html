使用媒體查詢創建響應式佈局
        將 CSS 網格與使用媒體查詢結合使用，
        如使用媒體查詢重新排列網格區域、更改網格尺寸以及重新排列網格項位置，
        我們可以讓製作出的網站更具響應性。
        
        在右側的預覽區中，當網頁可視區域的寬不小於 300px 時，列數從 1 變爲 2。 
        並且，廣告（advertisement）區域會完全佔據左列。
        

        當網頁可視區域的寬不小於 400px 時，請讓 header 區域完全佔據最頂行，
        footer 區域完全佔據最底行。
