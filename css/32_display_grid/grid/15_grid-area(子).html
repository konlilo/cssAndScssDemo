使用 grid-area 創建區域模板
我們在上一次挑戰中學到的 grid-area 屬性還有另一種使用方式。 
如果網格中沒有定義區域模板，你也可以像這樣爲它添加一個模板：

item1 { grid-area: 1/1/2/4; }
這裏使用了你之前學習的網格線編號來定義網格項的區域。 
上例中數字代表這些值：

grid-area: 
(row起始/column起始/row終點/column終點)
horizontal line to start at / vertical line to start at / horizontal line to end at / vertical line to end at;
因此，示例中的網格項將佔用第 1 條水平網格線（起始）和第 2 條水平網格線（終止）之間的行，及第 1 條垂直網格線（起始）和第 4 條垂直網格線（終止）之間的列。
