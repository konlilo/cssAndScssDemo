使用關鍵字更改動畫定時器
        在 CSS 動畫裏，animation-timing-function 用來定義動畫的速度曲線。 
        速度曲線決定了動畫從一套 CSS 樣式變爲另一套所用的時間。 
        如果要描述的動畫是一輛車在指定時間內（animation-duration）從 A 運動到 B，
        那麼 animation-timing-function 表述的就是車在運動中的加速和減速等過程。
        有一些預定義的關鍵字可用於常見的選項。 比如，默認值是 ease，動畫以低速開始，然後加快，在結束前變慢。 
        其它常用的值包括 ease-out：動畫以高速開始，以低速結束；ease-in，動畫以低速開始，以高速結束；linear：動畫從頭到尾的速度是相同的。
        
      給 id 爲 ball1 和 ball2 的元素添加 animation-timing-function，ball1
      的屬性值爲 linear，ball2 的屬性值爲 ease-out。 它們的 animation-duration
      都爲 2 秒，注意觀察它們在開始和結束時的不同。
