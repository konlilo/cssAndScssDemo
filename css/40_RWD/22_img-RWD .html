使圖片自適應設備尺寸

用 CSS 來讓圖片自適應其實很簡單。 你只需要給圖片添加這些屬性:

    img {
        max-width: 100%;
        height: auto;
    }
    設置 max-width 值爲 100% 可確保圖片不超出父容器的範圍；
    設置 height 屬性爲 auto 可以保持圖片的原始寬高比。
    
        給 responsive-img 添加樣式規則，使其成爲響應式的圖片。 
        它不應該超出父容器（在本例中，即預覽窗口）的範圍，並保持寬高比不變。 
        添加代碼後，拖動瀏覽器窗口，看看圖片發生什麼變化。
